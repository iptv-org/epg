# srf.ch

https://srf.ch

### Download the guide

```sh
npm run grab --- --site=srf.ch
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/srf.ch/srf.ch.config.js --output=./sites/srf.ch/srf.ch.channels.xml
```

### Test

```sh
npm test --- srf.ch
```
