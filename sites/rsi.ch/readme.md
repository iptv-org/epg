# rsi.ch

https://rsi.ch

### Download the guide

```sh
npm run grab --- --site=rsi.ch
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/rsi.ch/rsi.ch.config.js --output=./sites/rsi.ch/rsi.ch.channels.xml
```

### Test

```sh
npm test --- rsi.ch
```
