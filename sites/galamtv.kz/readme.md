# galamtv.kz

https://galamtv.kz/channels/now

### Download the guide

```sh
npm run grab --- --site=galamtv.kz
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/galamtv.kz/galamtv.kz.config.js --output=./sites/galamtv.kz/galamtv.kz.channels.xml
```

### Test

```sh
npm test --- galamtv.kz
```
