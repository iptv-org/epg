# xumo.tv

https://www.xumo.tv/ _[Geo-blocked]_

### Download the guide

```sh
npm run grab --- --site=xumo.tv
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/xumo.tv/xumo.tv.config.js --output=./sites/xumo.tv/xumo.tv.channels.xml
```

### Test

```sh
npm test --- xumo.tv
```
