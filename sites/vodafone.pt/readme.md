# vodafone.pt

### Download the guide

```sh
npm run grab --- --site=vodafone.pt
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/vodafone.pt/vodafone.pt.config.js --output=./sites/vodafone.pt/vodafone.pt.channels.xml
```

### Test

```sh
npm test --- vodafone.pt
```
