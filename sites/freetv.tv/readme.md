# freetv.tv

https://freetv.tv/

### Download the guide

```sh
npm run grab --- --site=freetv.tv
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/freetv.tv/freetv.tv.config.js --output=./sites/freetv.tv/freetv.tv.channels.xml
```

### Test

```sh
npm test --- freetv.tv
```
