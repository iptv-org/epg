# cgates.lt

https://www.cgates.lt/televizija/tv-programa-savaitei/

### Download the guide

```sh
npm run grab --- --site=cgates.lt
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/cgates.lt/cgates.lt.config.js --output=./sites/cgates.lt/cgates.lt.channels.xml
```

### Test

```sh
npm test --- cgates.lt
```
