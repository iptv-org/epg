# hoy.tv

https://hoy.tv/

### Download the guide

```sh
npm run grab --- --site=hoy.tv
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/hoy.tv/hoy.tv.config.js --output=./sites/hoy.tv/hoy.tv.channels.xml
```

### Test

```sh
npm test --- hoy.tv
```
