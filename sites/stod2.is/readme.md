```sh
npm run grab --- --site=stod2.is
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/stod2.is/stod2.is.config.js --output=./sites/stod2.is/stod2.is.channels.xml
```

### Test

```sh
npm test --- stod2.is
```
