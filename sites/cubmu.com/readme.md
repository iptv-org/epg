# cubmu.com

https://cubmu.com/live-tv _[Geo-restricted]_

### Download the guide

```sh
npm run grab --- --sites=cubmu.com
```

### Update channel list

```sh
npm run channels:parse --- --config=sites/cubmu.com/cubmu.com.config.js --output=sites/cubmu.com/cubmu.com.channels.xml
```

### Test

```sh
npm test --- cubmu.com
```
