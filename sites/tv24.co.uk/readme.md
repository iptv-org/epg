# tv24.co.uk

https://tv24.co.uk/

### Download the guide

```sh
npm run grab --- --site=tv24.co.uk
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/tv24.co.uk/tv24.co.uk.config.js --output=./sites/tv24.co.uk/tv24.co.uk.channels.xml
```

### Test

```sh
npm test --- tv24.co.uk
```
