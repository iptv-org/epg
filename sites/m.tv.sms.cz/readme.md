# m.tv.sms.cz

https://m.tv.sms.cz/

### Download the guide

```sh
npm run grab --- --site=m.tv.sms.cz
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/m.tv.sms.cz/m.tv.sms.cz.config.js --output=./sites/m.tv.sms.cz/m.tv.sms.cz.channels.xml
```

### Test

```sh
npm test --- m.tv.sms.cz
```
