# webtv.sk

https://webtv.sk/archive

### Download the guide

```sh
npm run grab --- --site=webtv.sk
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/webtv.sk/webtv.sk.config.js --output=./sites/webtv.sk/webtv.sk.channels.xml
```

### Test

```sh
npm test --- webtv.sk
```
