# meuguia.tv

https://meuguia.tv/

### Download the guide

```sh
npm run grab --- --site=meuguia.tv
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/meuguia.tv/meuguia.tv.config.js --output=./sites/meuguia.tv/meuguia.tv.channels.xml
```

### Test

```sh
npm test --- meuguia.tv
```
