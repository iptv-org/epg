# musor.tv

https://musor.tv/

### Download the guide

```sh
npm run grab --- --site=musor.tv
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/musor.tv/musor.tv.config.js --output=./sites/musor.tv/musor.tv.channels.xml
```

### Test

```sh
npm test --- musor.tv
```
