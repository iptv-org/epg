```sh
npm run grab --- --site=guidetnt.com
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/guidetnt.com/guidetnt.com.config.js --output=./sites/guidetnt.com/guidetnt.com.channels.xml
```

### Test

```sh
npm test --- guidetnt.com
```
