# tvkaista.org

https://www.tvkaista.org/

### Download the guide

```sh
npm run grab --- --site=tvkaista.org
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/tvkaista.org/tvkaista.org.config.js --output=./sites/tvkaista.org/tvkaista.org.channels.xml
```

### Test

```sh
npm test --- tvkaista.org
```
