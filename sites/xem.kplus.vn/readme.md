# xem.kplus.vn

https://xem.kplus.vn/

### Download the guide

```sh
npm run grab --- --site=xem.kplus.vn
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/xem.kplus.vn/xem.kplus.vn.config.js --output=./sites/xem.kplus.vn/xem.kplus.vn.channels.xml
```

### Test

```sh
npm test --- xem.kplus.vn
```
