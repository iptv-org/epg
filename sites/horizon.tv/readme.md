# horizon.tv

https://horizon.tv/

### Download the guide

```sh
npm run grab --- --site=horizon.tv
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/horizon.tv/horizon.tv.config.js --output=./sites/horizon.tv/horizon.tv.channels.xml
```

### Test

```sh
npm test --- horizon.tv
```
