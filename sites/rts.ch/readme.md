# rts.ch

https://rts.ch

### Download the guide

```sh
npm run grab --- --site=rts.ch
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/rts.ch/rts.ch.config.js --output=./sites/rts.ch/rts.ch.channels.xml
```

### Test

```sh
npm test --- rts.ch
```
