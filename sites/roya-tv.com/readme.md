# roya-tv.com

https://en.roya.tv/live-stream

### Download the guide

```sh
npm run grab --- --site=roya-tv.com
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/roya-tv.com/roya-tv.com.config.js --output=./sites/roya-tv.com/roya-tv.com.channels.xml
```

### Test

```sh
npm test --- roya-tv.com
```
