# dens.tv

https://www.dens.tv/

### Download the guide

```sh
npm run grab --- --site=dens.tv
```

### Update channel list

```sh
npm run channels:parse --- --config=sites/dens.tv/dens.tv.config.js --output=sites/dens.tv/dens.tv.channels.xml
```

### Test

```sh
npm test --- dens.tv
```
