# zuragt.mn

https://www.zuragt.mn/

### Download the guide

```sh
npm run grab --- --site=zuragt.mn
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/zuragt.mn/zuragt.mn.config.js --output=./sites/zuragt.mn/zuragt.mn.channels.xml
```

### Test

```sh
npm test --- zuragt.mn
```
