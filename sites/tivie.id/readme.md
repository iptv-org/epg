# tivie.id

https://tivie.id

### Download the guide

```sh
npm run grab --- --site=tivie.id
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/tivie.id/tivie.id.config.js --output=./sites/tivie.id/tivie.id.channels.xml
```

### Test

```sh
npm test --- tivie.id
```
