```sh
npm run grab --- --site=programtv.onet.pl
```

```sh
npm run channels:parse --- --config=./sites/programtv.onet.pl/programtv.onet.pl.config.js --output=./sites/programtv.onet.pl/programtv.onet.pl.channels.xml
```
