# tvmustra.hu

https://www.tvmustra.hu/

### Download the guide

```sh
npm run grab --- --site=tvmustra.hu
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/tvmustra.hu/tvmustra.hu.config.js --output=./sites/tvmustra.hu/tvmustra.hu.channels.xml
```

### Test

```sh
npm test --- tvmustra.hu
```
