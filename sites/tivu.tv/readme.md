# tivu.tv

https://www.tivu.tv/

### Download the guide

```sh
npm run grab --- --site=tivu.tv
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/tivu.tv/tivu.tv.config.js --output=./sites/tivu.tv/tivu.tv.channels.xml
```

### Test

```sh
npm test --- tivu.tv
```
