# tvireland.ie

https://www.tvireland.ie/tv/

### Download the guide

```sh
npm run grab --- --site=tvireland.ie
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/tvireland.ie/tvireland.ie.config.js --output=./sites/tvireland.ie/tvireland.ie.channels.xml
```

### Test

```sh
npm test --- tvireland.ie
```
